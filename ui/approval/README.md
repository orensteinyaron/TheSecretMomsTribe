# Approval UI

Mobile-first React web app. Yaron's only daily touchpoint.

## Features (Planned)
- Preview content exactly as it will appear
- Approve / Edit / Reject each post
- Adjust posting time
- Target: 5-10 minutes per day

## Stack
- React (Vite)
- Supabase Auth
- Supabase Realtime for content_queue
