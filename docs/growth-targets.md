# SMT Growth Milestones & Targets

## Milestone Unlocks

| Milestone | Unlock |
|---|---|
| 1K followers (each) | Credibility baseline, start testing CTAs |
| 5K followers (IG) | Instagram paid partnership eligibility |
| 10K followers (each) | Brand deal territory, first product launch |
| 50K followers (IG) | Link in bio drives real traffic |
| 100K followers | Full monetization layer activation |

---

## Phase 1 Targets (Current)

### Week 1-2
- Reactivate both accounts (new bios, profile images, link-in-bio)
- 3 TikTok posts/day + 1 IG post/day (all AI-generated)
- Research Agent producing daily briefings
- Approval UI functional

### Month 1
- 100+ followers on each platform
- Identify top 3 performing content types
- Establish posting rhythm and optimize timing

### Month 2-3
- 1,000 followers on each platform
- First CTA tests (email list, waitlist for AI product)
- Content Generation Agent fully autonomous
