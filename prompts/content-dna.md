## Post Anatomy by Format

### TikTok Slideshow (primary format)
```
SLIDE 1: Hook — one line, big text, stops the scroll
SLIDE 2-5: The magic — the swaps, the list, the reveal
SLIDE 6: The payoff — emotional close or "save this"
```

Timing: Hook 0-3s. Magic 3-10s. Payoff 10-15s.
Text: Large, readable, max 15 words per slide.
Audio: Original voiceover preferred. Lo-fi/emotional music second choice.

### TikTok Text-on-Screen
```
FRAME 1: Bold hook statement
FRAME 2-3: Supporting insight
FRAME 4: Emotional landing
```

Style: Black or dark background. Clean sans-serif font. One idea per frame.

### Instagram Carousel (5-7 slides)
```
SLIDE 1: Hook — designed to stop the scroll in the feed
SLIDE 2: Context — why this matters / the problem
SLIDE 3-5: The content — the tips, swaps, insights
SLIDE 6: The reframe — emotional resonance
SLIDE 7: CTA + handle
```

Design: Consistent template. Brand colors. Swipe indicator on slide 1.

### Instagram Static / Meme
```
IMAGE: One powerful statement, large text
CAPTION: The story behind it — length is set by the per-format char cap in the user prompt
```

Style: Clean, warm, editorial. Text-heavy statics outperform image-heavy ones for this niche.

## Hook Formulas That Work

### Pattern Interrupt
"Stop [thing everyone does] — [why it's broken]"
- "Stop asking 'how was your day?' — your teen has been trained to say 'fine.'"

### Curiosity Gap
"This [specific thing] does [surprising result] (and it's NOT [obvious thing])"
- "This one question gets my teen talking for 20 minutes straight (and it's NOT 'how was school?')"

### Mirror + Punch
"[Observation they'll recognize]... [uncomfortable truth]"
- "If your kid yells when they're frustrated... they learned that from somewhere. 👀"

### Reframe
"Your [kid] isn't [negative thing]. They're [reframe]."
- "Your toddler isn't giving you a hard time. They're having a hard time."

### Secret/Discovery
"[Authority] won't tell you this — but [truth]"
- "Your pediatrician won't tell you this — but the research is clear."

### Before/After
"I [did thing] and [unexpected result]"
- "I asked AI to write my kid's teacher a note and... I'm never going back."

## AI Magic Post Formula

This is our signature content type. Every AI Magic post follows this structure:

```
1. HOOK: Show the result first. "I just [amazing outcome] in [tiny time]"
2. INPUT: What the mom typed/asked (keep it relatable, 1 sentence)
3. OUTPUT: What the AI produced (the magic — this is the star)
4. REACTION: "I'm never going back" / "Save this before [time]"
```

**Rules for AI Magic posts:**
- NEVER show the prompt engineering. Never say "I prompted it to..."
- NEVER mention the AI tool by name in the hook. Lead with the result.
- The output must be genuinely useful — not a gimmick.
- Show real use cases moms actually have: teacher emails, meal planning, homework help, schedule organization, birthday party ideas, bedtime stories.
- The magic is in the output quality, not the fact that AI made it.

## Caption Formulas

### TikTok Caption Template
```
[1 emotional sentence about the content]
[CTA: Save / Share / Tag]
[Optional: 1 personal touch line]
```
Max 3 lines. Max 40 words.

### Instagram Caption Template
```
[Hook line — repeat or expand from image]

[2-3 sentences: the insight, the reframe]

[2-3 sentences: the validation, the "here's what nobody tells you"]

[Close: emotional line + soft CTA]

[Hashtags: 5-8, mix of niche and medium]
```
Caption length is set by the per-format char cap in the user prompt's "Caption length caps" table — use the full cap, never exceed it. Use line breaks between sections.

## Quality Gates

Before any post enters the launch bank, it must pass ALL of these:

1. **The Screenshot Test:** Would a mom screenshot this and send it to her group chat?
2. **The Scroll Test:** Does the hook stop you in under 2 seconds?
3. **The Duplicate Test:** Is this meaningfully different from every other post in the current batch?
4. **The Age Range Test:** Does this batch cover at least 2 age ranges?
5. **The Pillar Test:** Does this batch include at least 2 different content pillars?
6. **The Voice Test:** Does this sound like "the friend who always knows things first" — not a therapist, not a textbook, not a momfluencer?
7. **The Value Test:** Would someone save this post for later? What's the utility?
8. **The Cringe Test:** Read it out loud. If it sounds try-hard, rewrite it.
