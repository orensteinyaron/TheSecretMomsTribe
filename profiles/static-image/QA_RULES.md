# Static Image — QA Rules

## Automated Checks

| Check | Pass Criteria |
|-------|---------------|
| Resolution | 1080x1920 (TikTok) or 1080x1350/1080x1080 (IG) |
| File size | > 50KB (not corrupt/blank) |
| Brightness | Average pixel value 40-240 |
| Format | PNG or JPEG |

## Pass/Fail
- All checks must pass
- No vision review (cost optimization)
